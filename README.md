# Sentiment Analysis on IMDb Movie Reviews
## IMDb 영화 후기 감성 분석

**영화 후기에 쓰인 문장이 긍정적인지 부정적인지를 구별하는 모델 만들어보기**

분석을 위해 쓰는 것들
- IMDb dataset
- Pytorch와 TorchText
- Standard RNN architecture (첫번째 시도)
- Long Short-Term Memory architecture (두번째 시도)
